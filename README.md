# AI Powered Placement Platform

A web platform designed to help students prepare for placements by managing their profiles, resumes, skills, and academic information. The system also allows faculty and training & placement officers (TPO) to monitor and guide students.

## Features

* Student multi-step signup (education, experience, projects, skills)
* Resume upload
* Faculty and TPO registration
* Role-based login system
* Protected dashboards for Students, Faculty, and TPO

## Tech Stack

* **Frontend:** React.js
* **Backend:** FastAPI
* **Database:**  Mongo 
* **Other Tools:** Axios, React Router

## Run the Project

Install dependencies:

npm install

Start frontend:

npm run dev

Run backend (FastAPI):

uvicorn app.main:app --reload
